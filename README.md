# FallingPipeRender_Java

Interactive LWJGL scene for a pipe falling off a table.

The app reuses the original table, floor, and pipe proportions from the old render code and uses the texture assets already in `Assets/Textures/`.

## Run

Use `run.bat` from the project root. It compiles the sources against `C:\Program Files\lwjgl-release-3.3.4-custom`, extracts the Windows native DLLs into `.natives`, and launches the demo.

## Controls

- `SPACE` restart the animation
- `1`, `2`, `3` switch camera views
- `R` reset the scene
- `ESC` exit
